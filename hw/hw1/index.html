A brief summary of Internet Protocols

A network protocol is a set of rules and conventions for communicating data between network devices. These rules include mechanisms for connecting between machines and devices as well as rules for how data should be formatted in packets to be sent through the internet.

Most network protocols all generally use packet switching to send and receive data in the form of packets. Packet switching is the process of taking data and dividing it into smaller pieces to be sent across the internet like mail being delivered.

The main Internet Protocol(IP) itself, integrates with other high level protocols such as UDP, TCP, FTP, and HTTP to provide additional functions for the internet.
